Corewar

Qu'est-ce que Corewar ? Corewar est un jeu très spécial qui consiste à opposer de petits programmes les uns contre les autres dans une machine virtuelle GitHub. Le but est de créer des champions qui se départageront dans une arène nommée Machine Virtuelle afin d'en prendre le contrôle. Le projet Corewar se divise en deux parties principales : L'Assembleur (ASM) : La partie asm est chargée du parsing d'un fichier assembleur et de la génération de code compréhensible pour la VM. Et La Machine Virtuelle (VM) : Une machine virtuelle qui exécute un programme écrit en langage assembleur. La VM héberge les combats entre chaque champion
