Epytodo

EpyTodo est un projet de première année du cursus Epitech qui a pour but d'initier les étudiants au développement web. L'objectif est de créer une application de TODO List, plus spécifiquement une API REST qui gère les opérations CRUD (Create, Read, Update, Delete) pour une application de liste de tâches. Le projet a été réalisé à l'aide des Node.js : Le but de ce projet est de créer une todo list dynamique.
