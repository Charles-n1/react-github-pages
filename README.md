# react-github-pages — Portfolio

## Lancer le projet

```bash
# 1. Cloner le repo
git clone git@github.com:Charles-n1/react-github-pages.git
cd react-github-pages

# 2. Installer les dépendances
npm install

# 3. Lancer en local
npm start

# 3. Arrêter le programme
ctrl + c
```

Ouvre [http://localhost:3000](http://localhost:3000) dans ton navigateur.

---

## Commandes disponibles

| Commande | Description |
|----------|-------------|
| `npm start` | Lance le serveur de dev en local |
| `npm run build` | Build de production dans `build/` |
| `npm test` | Lance les tests |
| `npm run deploy` | Déploie sur GitHub Pages |

---

## Déployer sur GitHub Pages

```bash
npm run deploy
```

Le site sera accessible sur `https://Charles-n1.github.io/react-github-pages`
